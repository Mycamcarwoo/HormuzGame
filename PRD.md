# 지뢰찾기 모바일 웹게임 - 제품 요구사항 문서 (PRD)

## 1. 개요

### 1.1 프로젝트명
지뢰찾기 모바일 웹게임 (Minesweeper)

### 1.2 프로젝트 설명
클래식 지뢰찾기 게임을 모바일 웹 환경에 최적화하여 제작합니다. 셀을 클릭하여 지뢰가 없는 영역을 안전하게 개척하고, 숨겨진 지뢰의 위치를 추론하여 모든 지뢰를 찾아내는 전략 게임입니다.

### 1.3 타겟 플랫폼
- 모바일 웹 브라우저 (iOS Safari, Android Chrome)
- 반응형 디자인으로 PC에서도 플레이 가능

### 1.4 타겟 사용자
- 클래식 지뢰찾기 게임을 좋아하는 사용자
- 스마트폰으로 캐주얼 게임을 즐기는 사용자
- 논리적 사고력을 훈련하려는 사용자
- 짧은 시간에 플레이할 수 있는 게임을 찾는 사용자

---

## 2. 제품 목표

### 2.1 핵심 목표
1. **클래식 재현**: 기존 지뢰찾기 게임의 핵심 요소 완벽 구현
2. **모바일 최적화**: 터치 친화적 UI로 모바일 환경에서 원활한 플레이
3. **빠른 로딩**: 3초 이내 시작하여 언제 어디서나 즉시 플레이
4. **접근성**: 무료로 플레이 가능, 별도 앱 설치 불필요

### 2.2 성과 지표
- 일일 활성 사용자 (DAU): 1,000명 목표
- 평균 플레이 시간: 5분 이상
- 완료율: 30% 이상
- 재방문율: 40% 이상

## 3. 핵심 기능

### 3.1 게임 플레이

#### 3.1.1 난이도 선택
**난이도 종류**:
1. **초급**: 9×9 그리드, 10개 지뢰
2. **중급**: 16×16 그리드, 40개 지뢰
3. **고급**: 30×16 그리드, 99개 지뢰

#### 3.1.2 셀 상태
**셀의 상태**:
1. **닫힘**: 아직 클릭되지 않은 상태
2. **열림**: 클릭되어 주변 지뢰 수가 표시된 상태
3. **깃발**: 지뢰로 표시된 상태
4. **물음표**: 불확실한 셀에 표시된 상태
5. **지뢰 폭발**: 지뢰를 클릭하여 게임 오버된 상태

#### 3.1.3 게임 조작
**좌클릭 / 탭**:
- 셀 열기
- 지뢰가 있는 경우 게임 오버
- 지뢰가 없는 경우 주변 지뢰 수 표시
- 0인 경우 주변 셀 자동 열기 (재귀)

**우클릭 / 긴 터치**:
- 깃발 표시
- 다시 클릭하면 물음표
- 다시 클릭하면 닫힘

**더블 클릭 / 길게 긴 터치**:
- 이미 열린 셀의 숫자가 주변 깃발 수와 일치하면 주변 셀 자동 열기

### 3.2 점수 시스템

#### 3.2.1 기본 점수
- **승리**: 1000점
- **패배**: 0점

#### 3.2.2 시간 보너스
- 빠를수록 높은 보너스
- 초급: 최대 500점
- 중급: 최대 700점
- 고급: 최대 1000점

#### 3.2.3 난이도 보너스
- 초급: ×1
- 중급: ×1.5
- 고급: ×2

#### 3.2.4 총점 계산
```
총점 = 기본 점수 + 시간 보너스 × 난이도 보너스
```

### 3.3 타이머 시스템

#### 3.3.1 타이머 시작
- 첫 번째 셀 클릭 시 시작
- 초단위로 표시
- 999초 이상은 표시하지 않음

#### 3.3.2 기록 저장
- 각 난이도별 최단 시간 기록
- LocalStorage에 저장
- 랭킹 표시

### 3.4 깃발 카운터

#### 3.4.1 남은 지뢰 표시
- 총 지뢰 수 - 깃발 수
- 음수일 경우도 표시 (깃발 초과)

#### 3.4.2 자동 업데이트
- 깃발 추가 시 감소
- 깃발 제거 시 증가

### 3.5 결과 화면

#### 3.5.1 승리 화면
- "🎉 승리!" 메시지
- 소요 시간 표시
- 점수 표시
- "다시하기" 버튼
- "난이도 선택" 버튼

#### 3.5.2 패배 화면
- "💥 게임 오버" 메시지
- 폭발한 지뢰 위치 표시
- 잘못된 깃발 위치 표시
- "다시하기" 버튼
- "난이도 선택" 버튼

## 5. 게임 규칙

### 5.1 승리 조건
- 모든 지뢰가 아닌 셀을 열면 승리
- 남은 셀이 모두 지뢰인 상태로 깃발 표시

### 5.2 패배 조건
- 지뢰가 있는 셀을 클릭하면 패배
- 게임 중단 불가능

### 5.3 깃발 사용
- 깃발 수는 지뢰 수 이상 사용 가능
- 깃발 수는 게임 승리 조건에 영향 없음
- 다만 깃발 초과 시 카운터가 음수로 표시

### 5.4 첫 클릭 안전
- 첫 클릭은 항상 안전 (지뢰가 아님)
- 첫 클릭 주변에 지뢰가 없도록 보장
