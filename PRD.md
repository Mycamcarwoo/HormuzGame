# 호르무즈 지로찾기 - 제품 요구사항 문서 (PRD)

## 1. 개요

### 1.1 프로젝트명
호르무즈 지로찾기 (Hormuz Spot Finder)

### 1.2 프로젝트 설명
전 세계 주요 전략적 위치 중 하나인 호르무즈 해협을 주제로 한 모바일 웹게임입니다. 플레이어는 다양한 지리적 단서를 통해 호르무즈 해협의 정확한 위치를 찾아내야 합니다. 교육적 가치와 게임 요소를 결합한 위치 추론 게임입니다.

### 1.3 타겟 플랫폼
- 모바일 웹 브라우저 (iOS Safari, Android Chrome)
- 반응형 디자인으로 PC에서도 플레이 가능

### 1.4 타겟 사용자
- 지리 관심사가 있는 사용자
- 교육용 게임을 찾는 학생/교사
- 위치 추론 게임을 좋아하는 게이머
- 세계 지리에 관심 있는 일반 대중

---

## 2. 제품 목표

### 2.1 핵심 목표
1. **교육적 가치 제공**: 세계 주요 전략 위치에 대한 이해 증진
2. **게임적 재미**: 추론 요소로 흥미 유발
3. **모바일 친화적**: 언제 어디서나 쉽게 플레이 가능
4. **접근성**: 무료로 플레이 가능, 별도 앱 설치 불필요

### 2.2 성과 지표
- 일일 활성 사용자 (DAU): 1,000명 목표
- 평균 플레이 시간: 5분 이상
- 완료율: 30% 이상
- 재방문율: 40% 이상

## 3. 핵심 기능

### 3.1 게임 플레이

#### 3.1.1 스테이지 1: 기본 단서
**내용**:
- 호르무즈 해협에 대한 기본 설명 제공
- "페르시아만과 오만만을 연결하는 해협"
- "세계 석유 수송의 20%가 통과하는 전략적 위치"

**플레이어 행동**:
- 제공된 단서를 읽고 대략적인 위치 추론
- 세계 지도에서 클릭하여 위치 추정

#### 3.1.2 스테이지 2: 지형 단서
**내용**:
- 호르무즈 해협의 지형적 특징 제공
- "좁은 해협으로 폭 약 54km"
- "이란과 아랍에미리트 사이에 위치"
- "호르무즈 섬이 위치"

**플레이어 행동**:
- 지형 단서를 통해 범위 좁히기
- 정확한 위치 선택

#### 3.1.3 스테이지 3: 역사/경제 단서
**내용**:
- 호르무즈 해협의 역사적, 경제적 중요성 제공
- "고대부터 해상 무역의 중심지"
- "석유 수출의 핵심 경로"
- "전략적 봉쇄 가능성"

**플레이어 행동**:
- 역사적/경제적 단서로 위치 확정
- 최종 위치 클릭

### 3.2 점수 시스템

#### 3.2.1 정확도 점수
- **완벽**: 100점 (반경 50km 이내)
- **우수**: 80점 (반경 100km 이내)
- **좋음**: 60점 (반경 200km 이내)
- **보통**: 40점 (반경 500km 이내)
- **낮음**: 20점 (반경 1000km 이내)

#### 3.2.2 시간 보너스
- 1분 이내: +50점
- 2분 이내: +30점
- 3분 이내: +10점
- 3분 이후: +0점

#### 3.2.3 힌트 사용 페널티
- 힌트 사용 시: -20점

### 3.3 힌트 시스템

#### 3.3.1 힌트 유형
1. **지도 범위 힌트**: 정답 지역을 하이라이트
2. **국가 힌트**: 주변 국가 표시
3. **도시 힌트**: 주요 도시 표시

#### 3.3.2 힌트 사용 제한
- 게임당 최대 3개의 힌트 사용 가능
- 각 힌트 사용 시 점수 페널티 적용

### 3.4 결과 화면

#### 3.4.1 결과 표시
- 플레이어 추정 위치와 실제 위치 표시
- 거리와 정확도 점수 표시
- 전체 점수 및 랭킹 표시

#### 3.4.2 교육적 피드백
- 호르무즈 해협에 대한 추가 정보 제공
- 왜 전략적으로 중요한지 설명
- 실제 지리적 위치를 다시 확인할 수 있는 링크
